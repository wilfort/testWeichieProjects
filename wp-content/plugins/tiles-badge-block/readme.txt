Add a badge to highlight content areas on your website with our new badge Gutenberg block.

== Description ==

A quick and simple way to add a badge to your website.

#### Requires the Gutenberg plugin: 
This block uses a few newer features that are currently only available within the [Gutenberg plugin](https://wordpress.org/plugins/gutenberg/).

== Installation ==
1. Upload the `tiles-badge-block` folder to your `/wp-content/plugins/` directory or alternatively upload the tiles-badge-block.zip file via the plugin page of WordPress by clicking 'Add New' and selecting the zip from your local computer.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Search for the block within the Gutenberg block editor.

== Frequently Asked Questions ==

= Can this block be used on my theme? =
Yes! You can use this block on any Gutenberg block-based, full site editing, or universal WordPress theme.
